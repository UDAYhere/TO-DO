# ✅ Simple To-Do App

This is a basic **To-Do List App** built using HTML, CSS, and JavaScript.  
It allows users to:

- ➕ Add new tasks  
- ❌ Delete existing tasks

A great starting point for learning DOM manipulation and event handling in JavaScript!
